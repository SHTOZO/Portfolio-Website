# Portfolio Website

A modern portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Responsive design
- Modern UI with animations
- Dark mode support
- Project showcase
- Skills section
- Contact information


## Technologies Used

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons

## Deployment

The site can be easily deployed to Vercel:

1. Push your code to GitHub
2. Import your repository to Vercel
3. Deploy!
